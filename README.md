# 🏋️‍♂️ Qimo Gym - SaaS de Gestión Deportiva
### Next.js 14 | App Router | Supabase | Prisma | Tailwind CSS

**Qimo Gym** es una solución integral (Software as a Service) diseñada para digitalizar la operativa de centros deportivos. Actualmente, la plataforma está en producción gestionando la actividad de **+200 socios reales**, automatizando procesos que antes eran manuales.



## 🚀 Características Principales
- **Dashboard Administrativo:** Visualización en tiempo real de ingresos, altas de socios y vencimientos.
- **Gestión de Pagos:** Sistema automatizado de control de membresías y alertas de deuda.
- **Autenticación Segura:** Implementada con Supabase Auth (Magic Links y Social Login).
- **Diseño Mobile-First:** Interfaz optimizada para que los dueños de gimnasios gestionen todo desde su celular.

## 🛠️ Stack Tecnológico
- **Frontend:** React 18 & Next.js 14 (App Router) para un rendimiento óptimo y SEO.
- **Estilos:** Tailwind CSS con un sistema de diseño propio para una UI limpia y profesional.
- **Base de Datos & Backend:** PostgreSQL alojado en **Supabase**.
- **ORM:** **Prisma** para una gestión de datos tipada y segura.
- **IA Integration:** Flujos de trabajo asistidos por IA para análisis de métricas de retención.

## 🏗️ Arquitectura Técnica
El proyecto utiliza una arquitectura de **Server Components** para minimizar el bundle de JavaScript en el cliente, mejorando drásticamente los Core Web Vitals (LCP y CLS). La base de datos está normalizada para soportar el crecimiento de usuarios sin degradación de performance.
